# Brand assets — Yide Digital

Pasta para arquivos da identidade visual da Yide Digital.

## Arquivos esperados

Salvar aqui (substituir os placeholders quando criados):

- `logo.svg` — logo completo (wordmark "Yide DIGITAL"). Preferencial em SVG.
- `logo.png` — fallback PNG com fundo transparente.
- `logo-light.svg` — versão para fundo claro (teal sólido).
- `logo-dark.svg` — versão para fundo escuro (branco ou teal claro).
- `mark.svg` — apenas o símbolo (X com laço), sem texto, para uso compacto.
- `favicon.svg` — versão simplificada do mark, 32x32 ou 16x16.
